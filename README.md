# React_Ecommerce

This project is a React-based e-commerce application built as part of my learning journey.
The main goal of this project was to practice core React concepts by building real features instead of simple demos.

## Features
- Product listing
- Wishlist functionality
- Todo list feature (implemented as a separate component)
- Add and remove items
- Data persistence using localStorage
- Basic CRUD operations
- Page navigation using React Router

## Technologies Used
- React
- JavaScript (ES6+)
- HTML
- CSS
- Redux Toolkit
- React Router
- JSON Server (for mock API)

## Project Structure
- Components are separated by feature (products, wishlist, todo, authentication)
- Redux is used for managing shared state
- Custom hooks are used for handling API calls
- localStorage is used to store user-specific data like wishlist items

## What I Learned
- Building reusable React components
- Managing state using hooks and Redux Toolkit
- Working with localStorage for data persistence
- Implementing CRUD functionality
- Structuring a React project properly
- Handling routing in a React application

## Note
This project was initially started with a Todo List to understand core CRUD logic.
Later, the same logic was extended to implement a Wishlist feature, showing how functionality can be reused and scaled for real-world use cases.
